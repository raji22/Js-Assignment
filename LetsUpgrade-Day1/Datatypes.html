Datatypes in javascript

  • number

    eg: var a=12;

  • string

    eg: var name="Sanjay";

  • boolean

    eg: var b=True;

  • Array

    eg: var fruits=['apple,'mango','grapes'];

  • Objects

    eg: var person1={name:"Ram",age:"24"}

  • undefined

    eg: var age;

  • Null

    eg: var a=null;
