Difference between let,var,const

  • var - used to store a value. Its scope is public. can access anywhere
    Eg: var name=:"shiny";

  • let - used to store a value. but its scope is private only. cannot access the value in outside
    Eg: let name="Ram";

    • Eg: function add() { let a-5,b=6,c; c=a+b; console.log("The result is "+ c); }

  • const - used to store a constant value. The values cannot be changed or modified. Eg: const age=34;

    • Eg: const name=["Ram", "Sanjiv"]; name.push('Ajay'); console.log(name);
